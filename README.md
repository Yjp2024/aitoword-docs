# AIToWord

AIToWord is an AI-powered document conversion platform for converting Markdown, LaTeX, Mermaid diagrams, formulas, and structured text into editable Word or Office-compatible documents.

Official website: https://www.aitoword.cn

## Features

- Markdown to Word conversion
- LaTeX to Word conversion
- Mermaid to Visio-style diagram conversion
- Formula OCR recognition
- AI-assisted document formatting
- WeChat Mini Program support
- Online document conversion without manual formatting

## Use Cases

AIToWord is useful for:

- Researchers converting LaTeX notes to Word documents
- Students organizing formulas and Markdown notes
- Teachers preparing editable teaching materials
- Engineers converting Mermaid diagrams
- Office users converting structured AI-generated content to Word

## Supported Conversions

| Input | Output |
|---|---|
| Markdown | Word DOCX |
| LaTeX | Word DOCX |
| Mermaid | Diagram / Visio-style document |
| Formula Image | Editable formula / Word |
| Plain Text | Word document |
